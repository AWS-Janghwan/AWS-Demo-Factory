# 데모 시나리오
 1. 사용자가 챗봇 데모 웹페이지에 접속

 2. 제공되는 샘플 질문이나 제조 관련 질문을 실행

 3. SQL 기반으로 MES 데이터를 검색하고 결과를 정리

 4. 챗봇 화면에 조회된 MES 데이터와 결과를 출력
