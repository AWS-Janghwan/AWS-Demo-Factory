## 데모 시나리오
1. 사용자가 챗봇 데모 웹페이지에 접속  

2. "여수 플랜트의 최근 장비 오류 사항은?"과 같은 질문을 자연어로 입력  

3. SQL 기반으로 데이터를 검색하고 결과를 정리  

4. 챗봇 화면에 조회된 데이터와 결과를 출력
